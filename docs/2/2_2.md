## 网络

### 网络-打开网口：

```
ip addr
```

主要网口："2: enp4s0f0"

    打开即可访问教育网，其他网口不要动

    除lo网口（loop回环）外，网口均为自动获取ip地址（如需再次设置参考gpt）

### 网络-正常访问墙内网络：

```
# 进入root模式
sudo -i
# 切换目录
cd /home/jzhang/bin
# 打开login文件，写统一身份认证账号密码（学号：密码）
nano login
# Ctrl+W写入Ctrl+X关闭，运行脚本
run ./login
```
